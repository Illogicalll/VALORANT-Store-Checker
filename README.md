# Valorant-Store-Checker

Program that allows you to check your in-game store without having to launch the game

# Required

- Python 3.x.x
- aiohttp
  - https://pypi.org/project/aiohttp/
- asyncio
  - https://pypi.org/project/asyncio/

# Usage

- Download ZIP
- Run main.py
- Enter Username and Password
